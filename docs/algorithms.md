# 优化算法说明

## 支持的算法

### 1. NSGA2 (非支配排序遗传算法)
- **特点**：经典多目标优化算法，全局搜索能力强，Pareto前沿多样性好
- **适用场景**：变量少（<10个），评估时间短的场景
- **参数**：
  - population_size: 种群大小（默认50）
  - max_iterations: 最大迭代次数（默认100）

### 2. MOPSO (多目标粒子群算法)
- **特点**：收敛速度快，参数少，实现简单
- **适用场景**：中等规模问题（10~20个变量），需要快速得到结果
- **参数**：
  - population_size: 粒子群大小（默认30）
  - max_iterations: 最大迭代次数（默认50）

### 3. Surrogate-based Optimization (代理模型优化)
- **特点**：使用机器学习模型预测仿真结果，大幅减少真实仿真次数
- **适用场景**：仿真时间长（>5分钟/次），变量多（20+）的场景
- **参数**：
  - initial_samples: 初始采样数（默认50）
  - max_iterations: 最大迭代次数（默认30）
  - model_type: 代理模型类型（默认gpr：高斯过程回归）

### 4. MOBO (多目标贝叶斯优化)
- **特点**：基于贝叶斯优化的多目标算法，样本效率极高
- **适用场景**：仿真非常耗时（>30分钟/次），变量少（<10个）的场景
- **参数**：
  - initial_samples: 初始采样数（默认20）
  - max_iterations: 最大迭代次数（默认50）

## 算法选择指南
| 仿真时间 | 变量数量 | 推荐算法 | 预期加速比 |
|---------|---------|---------|-----------|
| < 1分钟 | < 10个 | NSGA2/MOPSO | 1x |
| 1~5分钟 | 10~20个 | MOPSO/Surrogate | 2~5x |
| 5~30分钟 | 20~50个 | Surrogate | 5~20x |
| >30分钟 | <10个 | MOBO | 20~100x |

## 双线架构（仅Surrogate算法）
- **优化线**：运行优化算法，使用代理模型快速预测
- **训练线**：独立进程训练代理模型，实时热替换到优化线
- 优势：优化过程不中断，模型不断更新，精度越来越高
